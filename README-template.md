## My process

I started designing the mobile version, creating the header component, and then moving to the main body and the individual cards. I then adjusted the design for desktop and finally implemented the dark mode styles, and the function to change the mode from light to dark mode.

### Built with

- Semantic HTML5 markup
- Flexbox
- CSS Grid
- Mobile-first workflow
- Tailwind CSS
- [React](https://reactjs.org/) - JS library


### What I learned

This was my first project with React. I learned how to use components and to pass props from one component to another. I also learned to adapt Javascript to JSX and how to use conditional formating. I learned a lot about how to use basic React structure.

### Continued development

I want to continue to grow with React, and learn to efficiently use Hooks and other properties of the library. I want to be more comfortable writing the code and adpating it to my needs to create projects that run more smoothly and quickly.
